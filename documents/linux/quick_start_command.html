# 一、快速上手几个命令

# 修改密码

# passwd
Changing password for user root.
New password:

按照这个命令，输入新密码进行修改。

# 创建用户

# useradd dxzhangg

执行这个命令，一个用户就被创建了。它不会弹出输入密码之类的页面，就会直接返回。因为接下来需要自己调用# passwd dxzhangg来设置密码，再进行登录。

# 用户分组

创建用户的时候，并未指定加入哪个组，于是默认就会创建一个同名的组。
执行# useradd -h可查看此命令的使用参数。
如果想看更详细的文档，可以通过# man useradd获得。

# Linux里是 "命令行 + 文件" 模式。

通过命令创建的用户，是放在/etc/passwd文件里面。这是一个文本文件。可以通过cat命令，将里面的内容输出在命令行上。组的信息放在/etc/group文件中。

# cat /etc/passwd
root:x:0:0:root:/root:/bin/bash
......
dxzhangg:x:1000:1000::/home/dxzhangg:/bin/bash


# cat /etc/group
root:x:0:
......
dxzhnagg:x:1000:

看到来root用户和新创建的dxzhangg用户。
x的地方是密码，密码当然不能放在这里，不然谁都可以知道。
接下来是用户ID和组ID，这和/etc/group里面对应。
/root和/home/dxzhangg分别是root用户和dxzhangg用户的主目录。
主目录登录进去后默认的路径。
/bin/bash的位置是用于配置登录后的默认交互命令行。

# 浏览文件

# cd

使用# cd 路径来切换目录。
# cd .表示切换到当前目录。
# cd ..表示切换到上一级目录。

# ls

# ls列出当前目录下的文件。
# ls -l用列表的方式列出文件。

drwxt-xr-x 6 root root    4096 Oct 20  2017 apt
-rw-r--r-- 1 root root     211 Oct 20  2017 hosts

第一个字符是文件类型。如果是 "-"，表示普通文件；如果是d，表示目录。当然还有很多种类文件类型。
第一个字符后面剩下的9个字符是模式，其实就是权限位。3个一组，每组rwx表示 "读（read）" "写（write）" "执行（execute）"。如果是字母，就说明有这个全县；如果是横线，就是没有这个权限。

这三组分别表示文件所属的用户权限、文件所属的组权限以及其他用户的权限。
例如：-rw-r--r--，表示这是一个普通文件，对所属用户，可读可写不能执行；对所属用户组，仅可读；对其他用户，仅可读。

# chmod

使用chmod 711 hosts可以改变权限。

# 安装软件

# 安装

CentOS：# rpm -i 文件路径。
Ubuntu：# dpkg -i 文件路径。
其中-i就是install的意思。

# 查看安装的软件列表

CentOS：# rpm -qa。
Ubuntu：# dpkg -l。

# 管道与搜索工具grep

执行# rpm -qa | grep jdk,这个命令是将列出来的所有软件形成一个输出。|是管道，用于连接两个程序，前面rpm -qa的输出就放进管道里面，然后作为grep的输入，grep将在里面进行搜索带关键词jdk的行，并且输出出来。grep支持正则表达式。
同理# dpkg -l | grep jdk也是样的结果。

# more和less

使用# rpm -qa | more和# rpm -qa | less这两个命令，可以将很长的结果分页展示出来。 more是分页后，只能往后翻页，翻到最后一页自动结束返回命令行，less是往前往后都能翻页，需要输入q返回命令行，q就是quit。

# 删除

CentOS：# rpm -e package。 -e就是erase。 Ubuntu：# dpkg -r package。 -r就是remove。

# 软件管家

CentOS软件管家：yum
Ubuntu软件管家：apt-get

CentOS下载源配置文件：/etc/yum.repos.d/CentOS-Base.repo
Ubuntu下载源配置文件：/etc/apt/sources.list

主执行文件在：/usr/bin或/usr/sbin
其他的库文件在：/var 配置文件在：/etc

还有也可以直接下载安装好的包：
使用wget下载，后面加链接，就能下载。
如果是zip包，则需要先安装以下程序。

# yum install zip.x86_64 unzip.x86_64
# apt-get install zip unzip
